Welcome to the Paragraph Tag Example

This is a simple paragraph. The <p> tag is used to define a block of text or paragraph in an HTML document. It automatically adds space before and after the paragraph.

HTML paragraphs are block-level elements, which means they always start on a new line, and browsers automatically add some margin (spacing) before and after each paragraph.
